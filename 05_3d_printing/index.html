Importantly, given the thickness of the acryllic, it's necessary to hold the motor at a certain angle so that the screws do not hit the motor and can run smoothly. <br> <h4> Programming the Microcontroller </h4> Now that the motor was attached to the structure, it was time to program the motor to rotate such that the seesaw would move up and down! I drew largely from the code we used in class, which was incredibly helpful. Here is what my code looks like: <br> <br> <code> // Thank you to the teaching staff of PS70 for providing the foundations of this code! <br> // I changed numbersto fit the motor and my structure, but the bulk of this code comes from our class website. <br> <br> const int A1A = 3; // define pin 3 for A-1A (PWM Speed) <br> const int A1B = 4; // define pin 4 for A-1B (direction) <br> <br> void setup() { <br> pinMode(A1A, OUTPUT); // specify these pins as outputs <br> pinMode(A1B, OUTPUT); <br> analogWrite(A1A, 0); // start with the motors off <br> digitalWrite(A1B, LOW); <br> } <br> <br> void loop() { <br> // start the motor at a low speed <br> analogWrite(A1A, 255); <br> digitalWrite(A1B, LOW); <br> delay(4000); // allow the motor to run for 4 seconds <br> <br> // stop the motor <br> analogWrite(A1A, 0); // setting both pins LOW stops the motor <br> delay(1500); // keep the motor off for 2 seconds <br> <br> // start the motor at a low speed in the other direction <br> analogWrite(A1A, 0); <br> digitalWrite(A1B, HIGH); <br> delay(4000); // allow the motor to run for 4 seconds <br> <br> // stop the motor <br> analogWrite(A1A, 0); // setting both pins LOW stops the motor <br> digitalWrite(A1B, LOW); <br> delay(1500); // keep the motor off for 2 seconds <br> }<br> </code> Thanks to [this link](http://web.simmons.edu/~grabiner/comm244/weekfour/code-test.html#:~:text=You%20can%20include%20code%20examples,code%20as%20what%20it%20is.) for help with showing code as <code> code</code> in html. The comments are very helpful, but the first two lines of code ensure that pin 3 of the microcontroller is connected to A-1A of the L9110 and pin for is connected to A-1B of the L9110. The next few lines of code, which define pins A-1A and A-1B as outputs, ensure that those pins will be doing what the Arduino tells them to do. Starting with A-1A at 0 (as A-1B is set to LOW) ensures that the motor is off to start. The next batch of code ensures that the motor is running at full power (the 255 number illustrates that) for 4 seconds (the delay(4000)) line runs this for 4000 milliseconds. Next, setting A-1A to 0 gives no power to the motor, stopping it for 1500 milliseconds. Then, to reverse direction, we set digitalWrite for A-1B to HIGH and want to move the motor at full power in the other direction. Now, though, because A-1B is set to HIGH, we put analogWrite for A-1A to 0 (255-255) and delay (4000) to run the motor for 4000 milliseconds. Finally, to stop the motor, we set digitalWrite for A-1B back to LOW and analogWrite for A-1A to 0, which will give no power to the motor. We set delay (1500) to run the motor for 1500 milliseconds. After this process is complete, the loop continues! (Thanks to [this link](https://www.circuitbasics.com/using-loops-in-arduino-programming/#:~:text=Loops%20are%20used%20to%20control,an%20iteration%20of%20the%20loop.) for help with understanding that loops continue over and over). <h4> Connecting the Wires from the L9110 to the Microcontroller to Arduino </h4> To really make the seesaw work, it was important to connect the wires! Like the code explains, I connected the VCC port of the L9110 to the 5V port in the microcontroller, the GND port of the L9110 to the GND port of the microcontroller, the A-1A port of the L9110 to the 3 port of the microcontroller, and the A-1B port of the L9110 to the 4 port of the microcontroller. <div class="row"> <div class="col-sm-4"> ![Motor Components](./motor_components.png) </div> </div> After connecting the L9110 to the Micro controller to my computer (the power source), we can see the seesaw work! <br> <video width="720" height="480" controls> <source src="seesaw_moving_final.mov" type="video/mp4"> </video> Given the strength of the motor required to move the seesaw, I found it necessary to hold the motor very tightly and in a position where the screws and the nuts do not get in the way of the motor and foundation pieces, respectively.
